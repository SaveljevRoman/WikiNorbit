Для получения x-api-key.
 
1) Переходим в контейнер с rightway_external_api docker exec -it rightway_external_api_1 bash
2) Переходим в папку rightway/rightway-external-api (хз по su apps или нет)
3) Выполняем команду php bin/console external-api:external-service-manager --env=prod
4) Нажимаем 0, т.к. хотим создать новый сервис
5) Вводим имя (какое хотим)
6) Вводим идентификаторы брендов (1 2 3 4 6 7 8 10 36 40 74)
7) Ввести ключ для доступа к api (кириллические символы запрещены) - это и есть x-api-key, его нужно придумать самому
8) Выбрать тип внешнего сервиса - выбираем касса 1С (4)
9) Выбрать роль внешнего сервиса - 0 (для read-only)
10) Вписать через пробел доступные версии - пишем 2.0
11) На первый вопрос отвечаем «yes», остальные «no»

P.S. созданные ключи находятся в таблице external_service
